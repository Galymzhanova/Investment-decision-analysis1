# Investment-decision-analysis
# Project Description:This project analyzes the relationship between the stock returns of McDonald's (MCD) and Coca-Cola (KO) ,the S&P 500 index (GSPC). Weekly historical data from Yahoo Finance was used to calculate returns and perform regression analysis in Excel.
#  Data Source:The data was downloaded from Yahoo Finance using weekly historical prices for:- McDonald's (MCD) -Coca-Cola (KO) - S&P 500 Index (GSPC)
# Tools Used- Google Spreadsheets - Yahoo Finance- Regression Analysis
# Analysis Steps 1. Downloaded weekly historical data from Yahoo Finance 2. Calculated weekly returns 3. Compared stock performance with the market index 4. Conducted regression analysis 5. Interpreted the results
